anycli-dev manifest
===================

helpers for anycli CLIs
# Commands

* [anycli-dev manifest](#manifest)
## manifest

generates plugin manifest json

```
USAGE
  $ anycli-dev manifest [PATH] [OPTIONS]

ARGUMENTS
  PATH  [default: .] path to plugin

OPTIONS
  --help     show CLI help
  --version  show CLI version
```
